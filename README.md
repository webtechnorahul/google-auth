🔐 Google Authentication (Node.js + Passport)

This project demonstrates how to implement Google OAuth 2.0 authentication using Node.js, Express, and Passport.js.

🚀 Features
Login with Google
Secure OAuth 2.0 authentication
Session-based login
User profile access
Logout functionality
🛠️ Tech Stack
Node.js
Express.js
Passport.js
Google OAuth 2.0
dotenv
